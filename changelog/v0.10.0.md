# KTailctl v0.10.0

## Features

- Add filtering of peer list based on DNS name
- Add toggle for exit nodes in peer list
- Add button to copy Tailscale SSH command to connect to host
- Add different styles of tray icon for better integration in different desktop environments
- Add option to disable tray icon
- Add option to select tray icon style
- Hide Mullvad nodes from peer list in tray icon
- Add two separate menus for self-hosted and Mullvad exit nodes
- Group Mullvad nodes in tray menu by country
- Add country flags to Mullvad nodes in tray menu

## Fixes

- Sort peers by DNS name instead of hostname
- Show DNS name instead of hostname in peer list
